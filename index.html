MODULE-1

This module includes research with the combination of JSON(Structured Data) + RESTful web service (over HTTP Protocol) + SQL Cloud Database + SQLite client Database.

DESCRIPTION

The module-1 of the project involves research over the performance on how an IoT application shall behave incase of a lightweight database at the GateWay level. Module-1 especially involves the usage of

  • SQLite - Gateway level lightweight database
  • MySql - Cloud hosted database
  • RESTful API
  • JSON(Structured Data)

These combinations vary for module-2.
